# Overview

Why am I uploading my portfolio's source code publicly ?
My goal here is to showcase my Web Development skills, feel free
to open any issue if you find something I could potentialy work on.

## Quickstart

*Requirements : latest npm & nodejs versions*

If you wanna use my repository as a template for your own portfolio simply run this command :

```bash
git clone https://github.com/hunixcode/icedcoffee.git
```

**or**

download the *.zip file in the [releases](https://github.com/hunixcode/icedcoffee/releases) section if you don't want the git headers.
