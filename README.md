## Quickstart

If you wanna use my repository as a template for your own portfolio simply run this commands :

```bash
git clone https://github.com/hunixcode/icedcoffee.git
```

**or**

download the *.zip* in the [releases](https://github.com/hunixcode/icedcoffee/releases) section if you don't want the git headers.
